## 安装运行
```
$前端
$ git clone 
$ cd Vue-personal-Blog-front-end
$ npm install
$ npm run serve
```

**技术栈**

- vue 
  -  条件循环、条件判断、组件插槽、传参、事件总线等
- vue-cli 
  -  生成脚手架 
- vue-router  
  - 路由传参、路由守卫等
- axios  
  -  发起ajax网络请求 
- element-ui 
  - 登录验证、各种ui组件、比如分页、加载、输入框、轮播图等等
- vuex
  - 状态管理

- vue-aplayer
  - 第三方音乐组件
